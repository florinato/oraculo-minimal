Política de Privacidad

1. Introducción

Bienvenido a ARCANA TAROT, una aplicación dedicada a proporcionar lecturas de tarot gratuitas. Esta Política de Privacidad explica cómo manejamos tu información cuando utilizas nuestra aplicación.

2. Información que Recopilamos

ARCANA TAROT no recopila, almacena ni accede a información personal sensible. Nuestra aplicación está diseñada para proteger tu privacidad al máximo.

  • Autenticación: Utilizamos el SDK de Pi Network para manejar la autenticación de forma segura y anónima. Pi Network es responsable de la protección de datos durante este proceso.
  • Datos de Sesión: No almacenamos historial de lecturas, preferencias personales ni ningún dato de comportamiento.
  • Información de Dispositivo: No accedemos a información del dispositivo como contactos, ubicación, fotos o archivos.

3. Modelo de Negocio

ARCANA TAROT es completamente gratuita para el usuario. Nuestro modelo de financiamiento se basa exclusivamente en anuncios integrados de Pi Network (Pi Ads). Estos anuncios son necesarios para mantener la aplicación funcionando de forma gratuita.

Al seleccionar las cartas para tu lectura, verás un anuncio intersticial de Pi Network antes de obtener tus resultados. Este modelo nos permite ofrecerte una experiencia completamente gratuita.

4. Cookies y Tecnologías Similares

Nuestra aplicación no utiliza cookies tradicionales ni tecnologías de rastreo. El navegador puede almacenar datos locales mínimos necesarios para el funcionamiento correcto de la aplicación, como preferencias de idioma.

5. Seguridad con Pi SDK

Utilizamos el SDK de Pi Network para autenticación segura. Pi Network implementa protocolos de seguridad robustos para proteger tus datos. Para más información sobre cómo Pi Network protege tus datos, consulta su Política de Privacidad oficial.

6. Derechos del Usuario

Tienes derecho a:

  • Acceder a cualquier información que puedan tener sobre ti
  • Solicitar la eliminación de tu información
  • Contactarnos con preguntas sobre tu privacidad

7. Cambios en Esta Política

Nos reservamos el derecho de actualizar esta Política de Privacidad en cualquier momento. Te notificaremos sobre cambios significativos mediante un aviso en la aplicación.

8. Contacto

Si tienes preguntas sobre esta Política de Privacidad o sobre cómo manejamos tu información, por favor contáctanos a través de la aplicación.
